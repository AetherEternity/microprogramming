что бы создать ваш шрифт, вы можете открыть index.html для нарисования букв.
Откройте fontText.txt выставте пераметры ширины и высоты:
после FONT_DATA: запишите вашь шрифт в формате:
(буква)
(битКодБУквы)

ПРИМЕР:
a
01110
00001
00001
01111
10001
10001
01111
b
10000
10000
10110
11001
10001
10001
11110

буквы и символы можно писать в любом порядке. С символами из ascii точно всё будет хорошо, с другими пока не тестил

После исполлните эту команду:
python3 ./makeFont.py
в данной папке появится font.tf,шрифт готов

Состав Шрифта:
в 0 байте хранится ширина букв
в 1 байте хранится высота букв
в 2 байте хранится информация сколько места занимает одна буква в байтах

далее
(номер буквы в ascii) * (сколько места занимает одна буква) = хранится начало бит карты буквы
